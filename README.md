# Uninformed Search in a Grid Environment

**Description:**
Visual AI Pathfinder demonstrating BFS, DFS, UCS, DLS, IDDFS, and Bidirectional Search on a grid with static and dynamic obstacles. Step-by-step GUI visualization using Pygame shows frontier nodes, explored nodes, and final path.

**Features:**

* Six uninformed search algorithms
* Dynamic obstacles with real-time re-planning
* Clockwise movement including diagonals
* Animated GUI titled `GOOD PERFORMANCE TIME APP`

---

## Installation

```bash
pip install pygame
```

---

## How to Run

```bash
git clone https://github.com/<your-username>/uninformed-search-grid.git
cd uninformed-search-grid
python main.py
```

---

## Usage Notes

* The GUI animates the search step-by-step.
* Dynamic obstacles may appear randomly; the path is re-planned automatically.
* Movement order follows clockwise rule with all diagonals.
